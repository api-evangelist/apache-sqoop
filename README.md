# Apache Sqoop (apache-sqoop)
Apache Sqoop is a tool designed for efficiently transferring bulk data between Apache Hadoop and structured data stores such as relational databases. It supports import and export of data with parallel processing capabilities.

## APIs

### Apache Sqoop
Sqoop provides a Java API and CLI for bulk data transfer between Hadoop (HDFS/Hive/HBase) and relational databases, with a connector API for custom data sources and sinks.
